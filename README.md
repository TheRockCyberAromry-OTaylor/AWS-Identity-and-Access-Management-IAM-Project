##  How to Log In or Create an AWS Account Using the Homepage Diagram
Use a preferred web browser and type in the URL https://console.aws.amazon.com

### 🖼️ Step 1: Locate the Sign-In Area
At the **top right corner** of the homepage, you'll see two key options:
- **Sign In**
- **Create an AWS Account**

These are your entry points to the AWS Console.

---

### 🔐 Step 2: Logging In (For Existing Users)
If you already have an AWS account:
1. Click **Sign In**.
2. You’ll be taken to the AWS Console login page.
3. Enter your **email or account ID**, then your **password**.
4. If Multi-Factor Authentication (MFA) is enabled, complete that step.
5. Once authenticated, you’ll land in your AWS Management Console.

🧠 *Tip*: Bookmark the Console URL for faster access next time: [https://console.aws.amazon.com](https://console.aws.amazon.com)

---

### 🆕 Step 3: Creating a New AWS Account
If you're new to AWS:
1. Click **Create an AWS Account**.
2. Fill out the registration form:
   - **Email address**
   - **Password**
   - **AWS account name** (this can be your company or personal name)
3. Choose your **account type**: Personal or Professional.
4. Enter **billing information** (credit/debit card required—even for free tier).
5. Verify your identity via **SMS or phone call**.
6. Select a **support plan** (you can start with the free Basic plan).
7. Once complete, you’ll be redirected to the Console.

🎉 *You now have access to the AWS Free Tier*, which includes services like EC2, S3, and Lambda at no cost for 12 months.
